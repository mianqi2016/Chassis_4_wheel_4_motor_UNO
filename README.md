# Chassis_4_wheel_4_motor_UNO
Chassis driver for 4-wheel-4-motor by MCU ATmega328P, written in Arduino C.

Hardware framework:
1. DC Motor
2. L298n
3. IR-counter
4. 8-Channel Remote Controller 2.4GHz

Hardware modules:
1. IR_counter
2. ADXL345
3. HC-SR04
4. LCD1602
5. HC-05/06
6. nRF24L01
7. DS18B20
8. DHT11
9. DS1302/1307
10. ISD1820
11. OV7670
12. WS2812B
13. LCD1602
14. AT24C01C
15. NEO-6M

Software components:
1. PCINT
2. SoftwareSerial
3. EXTI
4. pulseIn()
