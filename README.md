# Chassis_4_wheel_4_motor_UNO
Chassis driver for 4-wheel-4-motor by MCU ATmega328P, written in Arduino C.

Hardware modules:
1. IR_counter
2. ADXL345
3. HC-05/06
4. nRF24L01
5. DS18B20
6. DHT11
7. DS1302/1307
8. ISD1820
9. OV7670
10. WS2812B
11. LCD1602
12. AT24C01C
13. NEO-6M

Software component:
1. PCINT
2. SoftwareSerial
3. EXTI
